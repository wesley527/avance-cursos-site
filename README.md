# Avance Cursos

Site da Avance Cursos.

## Como editar facil

- O projeto principal continua em React/TanStack Start.
- Para edicao simples, abra `site-editavel.html`.
- Textos, links e imagens ficam direto no HTML.
- As imagens ficam em `src/assets`.

## Rodar o projeto

```bash
npm install
npm run dev
```

## Gerar build

```bash
npm run build
```
